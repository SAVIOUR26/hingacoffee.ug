### Pages Plan
1. **Home** (`index.php`) — Hero, stats counter, video, gallery strip, CTA
2. **About** (`about.php`) — Company overview, vision/mission, values
3. **Products** (`products.php`) — Green beans, roasted, ground, custom blends
4. **Gallery** (`gallery.php`) — Masonry grid of all 15 images with lightbox
5. **Sustainability** (`sustainability.php`) — Farmer empowerment, eco practices
6. **Contact** (`contact.php`) — Form, address, map

### Tech Stack
- PHP (no framework, clean procedural includes)
- Vanilla CSS (custom properties, Grid, Flexbox)
- Vanilla JS (Intersection Observer, lightbox, mobile nav)
- Google Fonts: Playfair Display + Inter
- Font Awesome 6 (icons)
